VectorNav Python Library¶

The VectorNav Python Library (vnpy) provides easy access to VectorNav’s sensors.

  • Version History
  • Known Issues
  • Library API Documentation

Installation¶

Before taking a look at the examples, Python and vnpy will need to be installed.

  1. First make sure you have 32-bit version of Python 3.4 or greater installed and callable from the command line.
  2. Open a terminal and change directory to vnproglib-X.X/python.
  3. For Windows, execute the command python setup.py install. For Linux, execute the command sudo python3 setup.py install.

Troubleshooting

If the installation has failed here are a few common issues.

  1. If you recive a failure message stating that a Visual Studios component is missing or has failed, reinstall Visual Studios and make sure that the following items are selected during installation.
  2. If the VectorNav Programming Library has successfully installed, then there should be a vnpy folder in your Python program folder. If no folder is present, you will experince a path issue when trying to run the programming library code. Your program needs to have visibility to the location of the VectorNav Programming library. I would check within the project's site package information if the vnpy package is visible. You can also check that there is a vnpy folder in your Python program data folder (Python -> Lib -> site-packages -> vnpy).

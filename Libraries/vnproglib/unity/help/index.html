VectorNav Unity Library

The VectorNav Unity Library is geared towards development for using the Unity game engine.

To get started, we recommend first going through the Getting Started example below to get comfortable with connecting and interacting with a VectorNav sensor. Following this example, the others listed are useful areas to jump into.

  • Getting Started
  • From Scratch

After you work through the examples, since access to VectorNav sensors within Unity is largely performed by importing and access the source files of the VectorNav .NET Library, it is recommended that you refer to the VectorNav .NET Library documentation for advanced usage and configuration.

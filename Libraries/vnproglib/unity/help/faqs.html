How do I connect a VectorNav sensor to an Android phone?

Though there exist many different ways to connect a VectorNav sensor to a phone, the most straight forward method is to connect the USB connector attached to a sensor to the phone's USB port. Now the USB cable provided will obviously not fit into the standard USB port on the phone, this is because the provided sensor USB cable has a USB Standard Type A connector while the phone will need a USB Micro Type B connector. Appropriate adapters may be found by performing a web search for the terms "usb a to micro b adapter". If you have trouble finding a suitable adapter, please contact VectorNav at suppo.nosp@m.rt@v.nosp@m.ector.nosp@m.nav..nosp@m.com for a recommended supplier and part number.
